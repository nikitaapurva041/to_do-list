
To-Do List CLI Application (Python)

A simple Command-Line To-Do List application built using Python that allows users to manage daily tasks efficiently.

Features

➕ Add new tasks
❌ Delete existing tasks
👀 View all tasks (with numbering)
✏️ Update tasks
✅ Mark tasks as completed
🔁 Interactive menu system
🧠 User-friendly input handling

Tech Stack

Python 3
Built-in libraries only (no external dependencies)

Project Structure

todo-app/
│
├── app.py        # Main application file
└── README.md     # Project documentation

2️⃣ Run the program
python app.py

How It Works

After running the script, you will see a menu like:

To Do List Menu:
1. Add the task
2. Delete the task
3. View the task
4. Update the task
5. Mark the task as completed
6. Exit

Available Options
  Option	Action
    1	Add a new task
    2	Delete a task
    3	View all tasks
    4	Update an existing task
    5	Mark task as completed
    6	Exit the program
